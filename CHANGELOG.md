# Changelog

## 0.0.1

Initial scaffold for AiLang VS Code support.

- Adds `.aos` and `.aiproj` language registration.
- Adds TextMate grammar scaffold for AiLang node syntax.
- Adds snippets for common AiLang forms.
- Adds editor language configuration.

Semantic validation, formatting, evaluation, and diagnostics remain owned by the AiLang toolchain.
